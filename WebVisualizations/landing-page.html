Summary vs Latitude X

This website shows the analysis results of data comparing the latitudes of a randomly selected set of 500 cities to various weather phenomena including cloudiness, humidity, temperature and wind speed.


The dataset of 500 cities was assembled using data pulled from the OpenWeatherMap API. Matplotlib was used to create plots comparing each weather phenomena to the city's latitude. This site provides the source data used as well as each plot and descriptions of trends and correlations within the data.
